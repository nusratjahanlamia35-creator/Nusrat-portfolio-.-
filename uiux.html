Step 01

Project Overview

Project goal, target users and success metrics at a glance.

Case Study · Personal Portfolio
Nusrat Web Studio — Freelance Portfolio
A high-converting personal portfolio designed to attract premium clients. The goal: make every visitor feel like they're talking to a senior expert — not just browsing a resume.
Goal: Generate Leads Users: Business Owners Timeline: 1 Week
Project Goal
Convert portfolio visitors into paying clients through trust, credibility and a clear "Hire Me" call-to-action.
Target Users
Small business owners, startup founders and agency managers who need fast, premium web design.
Success Metric
Visitors click "Hire Me" or send a message within 3 minutes of landing on the page.
Step 02

Problem Statement

What problem we are solving and what the user pain points are.

Boring Portfolios
Most freelancer portfolios look flat, generic and fail to build instant trust with potential clients.
No Clear CTA
Visitors don't know what to do next — no clear "Hire Me" path leads to missed opportunities.
Not Mobile Friendly
Over 60% of users browse on mobile — a non-responsive portfolio loses more than half its audience.
Slow Load Time
Heavy portfolios with bloated code and unoptimized assets lose visitors in the first 3 seconds.

Step 06

Key Screens

Layout structure of each main section in the portfolio.

Hero Section
Left: Bold headline + description + CTA buttons. Right: Profile/illustration. Big spacing, high contrast.
Services Section
3-column grid of glass cards. Each card: icon + title + 1-line description. Center-aligned header.
About Section
2-column: Left = photo, Right = intro text + key highlights. Personal, professional tone.
Contact Section
Centered form: Name + Email + Message + Submit. Clean glass card, Formspree integrated.
